# Dataset Instructions

Suggested dataset structure for `recycle_dataset/`:

recycle_dataset/
├── plastic/
│   ├── img1.jpg
│   └── ...
├── glass/
├── metal/
└── paper/

Tips:
- Aim for >= 200 images per class for reasonable baseline accuracy.
- Use data augmentation in the notebook for better generalization.
- You can use Kaggle to find recyclable waste image datasets or create your own.
